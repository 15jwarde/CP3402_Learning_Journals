## Content Insights
I learned that while no code solutions like WordPress do provide extreme ease, it does limit users trying to implement extremely specific UI. For example my scrollable area for fan favourites. In VS Code, I had to manually write “flex-wrap: nowrap” to create a horizontal scroll, but WordPress kept trying to force my squares to fit in the row area. To combat this I had to implement custom CSS to help maintain their size.  

## Career/Employability/Learning Insights
Building the same site across three different platforms (HTML/CSS, Joomla, and now WordPress) has given me a significant career insight into platform versatility. I’ve also realized that my learning is most effective when I reverse-engineer. By taking a finished design I already built and forcing WordPress to replicate it, I learned much more about the Block Editor's limitations and strengths than if I had just used a pre-made template. This keeps me from becoming reliant on defaults and ensures I can meet a client's specific branding requirements regardless of the tools.

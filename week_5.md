## Content Insights
I learned that a Child Theme is the industry standard way to customize a CMS without modifying core files. 
I also explored Docker as a development environment since my group plans to use that local system for our assignment. I learned that by using a YAML file to define services, I can create a localized mini-server that includes its own database and web server. This is a significant step up from standard installers because it uses Volumes to map files directly from my hard drive into the container, allowing me to use my own text editor while the code runs in a Linux environment.

# Week 4
## Learning Activities & Resources
Advanced Custom Fields (ACF) Documentation: Researched how to create structured data fields for venues (Safety Ratings, Addresses) rather than relying on the standard post editor. <br>
Bulk Data Management: Learned how to use WP All Import to map CSV data columns to ACF fields to avoid manual data entry for multiple cities.

## Estimated Hours
7 hours

## Content Insights
I learned how to transform a Custom Post Types blog style to a structed directory using ACF. For example, the safety rating for a bakery in Sydney is a specific metadata value that can be queried and filtered, which is far more powerful than just typing the number into a paragraph. I also gained a deeper understanding of the migration process. Using a migration plugin taught me how the server automates search-and-replace functions to ensure localhost links are updated to the live domain during deployment.

## Career/Employability/Learning Insights
This week taught me the value of automation over manual labor. Using WP All Import to bulk-upload venue data from a CSV was a major efficiency win. In a professional setting, being able to manage large datasets quickly is a key skill. It allowed me to focus on the UI and functionality rather than spending hours on repetitive data entry. Developing locally allowed me to experiment with heavy plugins like Search & Filter and WP Ultimate Post Grid without the risk of breaking the live site.
